Sentiment Analysis of Stocks on r/Wallstreetbets

  • Motivation: Allow users to save time and quickly gain insights on current trending companies on the financial subreddit
  • Obtains comments from posts relating to the selected company via Reddit API
  • Preprocesses text data by removing irrelevant words, punctuations, conjunctions, etc.
  • Performs sentiment analysis of comments relating to the company using VADER and identifies top mentioned words
  • Deployed on Heroku as a web application where users can easily access
  • Visit Dashboard
  • Github

Supermarket Customer Segmentation

  • Motivation: Identify unique groups of customers based on demographics and behaviors to allow for increased cost-effectiveness of marketing and engagement campaigns
  • EDA on potential important features and interactions
  • Identified 4 groups of customers with the Elbow method using K-means clustering
  • Matched clusters and features to interpret the characteristics of each customer group
  • Visit Notebook

Mental Health Interactive Dashboard

  • Motivation: Visually compare the general situations of mental health issues for employees in the tech industry to determine relevant factors on improving current labor policies
  • Cleaned data by dealing with missing data and grouping responses for features with high cardinality
  • Presented insights with interactive plots where users can select and filter certain characteristics
  • Web application created with Altair and Dash
  • Visit Dashboard
  • Github

Credit Card Default Prediction

  • Motivation: As credit card defaults are costly to businesses, correctly and efficiently identifying users that will likely default will provide immense value
  • EDA of features and preprocessed data in preparation for modelling pipeline
  • Explored multiple machine learning algorithms such as Logistic Regression, K-Nearest Neighbours, Random Forests, and LightGBM
  • Used RandomizedSearchCV to tune hyperparameters of models resulting in a 0.69 F1-macro score on the test set
  • Interpreted results and feature importances of the best performing model
  • Visit Notebook

Traffic Collision Analysis with SQL

  • Performed complex SQL queries involving Joins, CTEs, etc.
  • Used Folium’s geospatial mapping to understand hotspots for traffic collisions
